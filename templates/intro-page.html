Chess AI Project

Welcome to the home page for the chess AI project. This is a project for learning how to develop different types of chess AI using python. From this page you can play against the different AIs as well as have the AIs play games against one another.

This project was developed by Dylan Crockett with the source code available at https://github.com/dylancrockett/play-chess-ai under the MIT license. Feel free to play against the AIs for fun or adapt some of the code for another project feel free!
